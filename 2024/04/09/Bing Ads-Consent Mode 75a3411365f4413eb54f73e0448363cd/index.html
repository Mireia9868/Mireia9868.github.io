Bing Ads-Consent Mode

Question 1: Is it necessary for us to request clients to install Consent Mode? If so, is there a deadline for implementation? Additionally, if we choose not to implement it, how will it affect our account traffic and data tracking?

Answer: Bing Ads Consent Mode is not mandatory, but it is a recommended feature that enhances the privacy capabilities of UET and allows you to control the storage of first-party and third-party cookies based on the user’s consent status.

This can help you comply with local regulations regarding data collection and manage consent. There is no deadline for implementation, but you can start using it at any time by adding the consent script to your UET tags. If you choose not to implement it, it will not have an impact on your account traffic and data tracking. However, you won’t be able to adjust UET cookie access based on the consent status.

Question 2: If our clients decide to install Consent Mode, can Microsoft provide support or offer detailed documentation for verification?

Answer: Currently, Microsoft technology cannot provide online support.

Official documentation: https://help.ads.microsoft.com/#apex/ads/en/60119/1

Verification process:

To verify the implementation of UET Consent Mode, you can use the UET Tag Helper extension for Chrome or Microsoft Edge browsers. The UET Tag Helper shows you the consent status of UET tags on each page of your website and the events being sent or blocked by UET based on the consent settings. To use the UET Tag Helper, follow these steps:

  • Install the UET Tag Helper extension from the Chrome Web Store or Microsoft Edge Add-ons Store.
  • Navigate to your website and click on the UET Tag Helper icon in the browser toolbar.
  • In the UET Tag Helper panel, you can see the UET tag ID, consent status, and the events being sent or blocked by UET.
  • You can also manually change the consent settings by clicking on the Consent Mode dropdown and selecting “Granted” or “Denied.” This will simulate how UET behaves when a user provides or denies consent on your website.
  • You can refresh the page or navigate to other pages to see how Consent Mode affects UET tags and events on different pages.
