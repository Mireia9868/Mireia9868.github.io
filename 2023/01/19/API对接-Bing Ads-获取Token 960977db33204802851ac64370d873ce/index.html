API对接-Bing Ads-获取Token

获取Token

  • 申请一个独立的 MS Account 作为 API user. 并请确保这个 USER 之前没有绑定到任何
    Microsoft Manage Account（CID）账户, 是一个全新的 User。
  • 邀请这个Account到客户的 CID 下
  • 访问 Develop center，并选择这个账户，点击 Request Token
    1. https://developers.ads.microsoft.com/Account
    2. 系统会自动生成 Universal Developer Token

使用 OAuth 进行身份验证

  • 注册应用程序

    • 注册位置https://portal.azure.com/#view/Microsoft_AAD_RegisteredApps/ApplicationsListBlade

    • 在“名称”部分中，输入一个有意义的应用程序名称，该名称将显示给应用程序的用户
    • 在“支持的帐户类型”部分中，选择“任何组织目录中的帐户和个人 Microsoft 帐户”
      • 在初始应用注册期间，必须选择任何组织目录中的帐户和个人 Microsoft 帐户。除非修改应用程序清单，否则以后无法在门户中更新此设置 "signInAudience"
    • 选择“注册”以创建应用程序。

    • 在应用的“概述”页上，找到“应用程序（客户端）ID”值，并将其记录下来供以后使用。当您请求用户同意并获取访问令牌时会用到

    https://learn.microsoft.com/en-us/advertising/guides/authentication-oauth-register?view=bingads-13

  • 请求用户同意您的应用程序管理他们的 Microsoft Advertising 账户

    • https://learn.microsoft.com/en-us/advertising/guides/authentication-oauth-consent?view=bingads-13

  • 获取访问和刷新令牌

    • https://learn.microsoft.com/en-us/advertising/guides/authentication-oauth-get-tokens?view=bingads-13

  • 首次调用 API

    • https://learn.microsoft.com/en-us/advertising/guides/authentication-oauth-quick-start?view=bingads-13

参考链接：
