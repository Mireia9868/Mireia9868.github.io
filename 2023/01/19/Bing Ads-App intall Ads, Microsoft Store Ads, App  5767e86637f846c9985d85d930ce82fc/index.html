Bing Ads-App intall Ads, Microsoft Store Ads, App Extension效果对比

App投放案例总结：

目前PC端的应用投放成功案例较多，即在PC端下载安装，PC端游玩或使用的游戏或应用

Search广告投放前需提前调查对应市场广告词搜索量，并可以配合Extension测试

Audience Image BaseAds表现整体比较好，通过Audience和publisher对人群进行定位，可以作为投放测试方向之一

App intall Ads & Microsoft Store Ads在help center有提供对接方案，可以通过MMP追踪到后端数据，需要客户提前对接上第三方数据平台进行测试。

广告适用范围对比：

Advertising Device 转化追踪 Network 投放条件 country App Store
Microsoft Store Ads - 3rd party API, UET SDK, Microsoft UI report Microsoft Store, Bing Search, Widgets on Microsoft Start 在MS store中上线产品 US Microsoft Store
App intall Ads¹ iOS & Android mobile 衡量URL和可选的第三方跟踪模板² 仅限文本 广告，适用于必应和雅虎拥有和运营的网站 客户与三方衡量平台（MMP）有合作 AU, BR, CA, FR, DE, IN, UK, US iTunes, Google Play, Windows Phone Store, Windows Store
App Extension¹ Mobile & Tablet 衡量URL和可选的第三方跟踪模板 仅限Text Ads，在必应上可用 - AU, BR, CA, FR, DE, IN, UK, US iTunes, Google Play, Windows Phone Store, Windows Store

¹目前该广告类型可以判断客户用户设备和操作系统，并针对性引流到对应应用商店

²App intall Ads当前支持第三方衡量平台：AppsFlyer、Kochava、Tune、Singular、Adjust 或 Branch

广告人群覆盖

Search Ads：

整体游戏相关流量较小，根据细分市场获取相关搜索量情况配合App Extensions使用

MSAN Ads：

MSAN image-base广告比较适合广告测试，可以更高效率复用迭代优质素材，整体流量也相对可观，推荐游戏类及App类测试

App intall Ads & App Extension：

App intall Ads：广告覆盖面较大，且可以通过第三方追踪到较后端的数据，适合有一定规模且有一定预算测试小渠道的客户

App Extensions：通过广告中明确的 "行动号召 "提高客户参与度。推动安装详细报告、 跟踪点击、应用程序安装和性能指标。通过智能检测自动检测用户设备和操作系统在广告系列或广告组级别管理扩展。

覆盖面：7 亿多次独立搜索；全球每月搜索次数超过 140 亿次；第二大搜索引擎；全球市场份额 18%

Microsoft Store Ads：

可以与 Microsoft Store 中搜索的高意向桌面用户建立联系，并使得用户发现、下载和参与您的应用程序和游戏，可以通过第三方追踪到较后端的数据，适合有一定规模且有一定预算测试小渠道的客户

覆盖面：全球 9 亿多商店独立访客；50% 的 Windows 新用户会在第一个月访问商店；Windows 11 使商店流量增加了 3 倍；52% 的用户年龄在 18-34 岁之间 覆盖年轻 人群
