微软广告2023年1-10月产品更新

2023-10更新：

1. 受众广告更新

。

Microsoft Audience Network 上的产品广告

  • 新的出价方式测试：最大化转化次数和目标每次转化费用
  • **新市场：（**已发布）：微软最近在 8 月中旬将 Audience Ads 扩展到了 58 个新市场，这使得其覆盖范围达到了全球 187 个市场。
  • 有价值的观众
  • **预测定位：（**已发布）：最近针对受众广告发布了预测定位，它允许广告商利用 Microsoft 受众智能来查找他们可能没有考虑过的更有可能转化的相关受众。
  • 更具吸引力的广告：
  • Microsoft 休闲游戏扩展（推出）：Microsoft 正在将受众广告扩展到新的展示位置，例如默认在 Windows 计算机上运行的游戏的主页、菜单和游戏屏幕。受众广告目前在 Treasure Hunt 和 Jigsaw 中投放，到 2024 年将在所有 Microsoft 休闲游戏中投放。
  • **Microsoft 365 扩展：（**推出）：Microsoft 正在将 Audience Ads 扩展到 Microsoft 365 移动应用的免费消费者版本的新市场。Android 版将于 10 月在美国推出，1 月在欧洲市场推出。
  • **视频和 CTV 广告：（**已发布）：微软发布了视频和 CTV 广告，使品牌可以通过在线视频和 CTV 展示位置接触 Microsoft 资产和发布商合作伙伴的大规模和高价值受众。

2. 旅行住宿的新 UI 增强功能

从10 月 3 日开始 ，酒店营销活动将更名为住宿营销活动，所有“酒店”术语将更新为“酒店”术语

UI 增强后

3. 更多垂直广告在搜索合作伙伴网络中推出：信用卡和房产促销广告

搜索合作伙伴网络中提供了更多垂直广告，汽车和旅游以及活动广告已经推出，但现在我们在搜索合作伙伴网络上推出了信用卡和房产促销广告。

更多垂类广告：通过垂直广告保持较高的效果。

2023-09更新：

1. Microsoft Advertising Editor 中的 Audiences已经支持批量关联

目前支持再营销、有购买意向的受众群体、动态再营销列表、类似受众群体、客户匹配、自定义受众群体和自定义组合列表的批量关联。

*但创建和管理受众仍应通过 Microsoft Advertising 在线完成，

但现在 Microsoft Advertising Editor 可用于批量更新关联。为此，请导航至**“受众”**选项卡并选择您所需的操作。您还可以通过导入文件或从 Google Ads 导入来更新受众群体定位关联在新窗口中打开。

此外，现在所有 Microsoft Advertising 市场（欧洲、中东和非洲、亚太地区和拉丁美洲）都可以找到有购买意向的受众！要查看市场和可用受众的完整列表，请访问有市场受众：吸引准备购买的人 (microsoft.com)在新窗口中打开。

2. 两项自适应搜索广告更新：自动生成的资源和创建多媒体广告

Microsoft Advertising 中全面发布了用于自适应搜索广告 (RSA) 的自动生成资源。

当您在广告活动设置中启用此功能时，除了您提供的Ads set之外，还开始提供自动生成的资产，从而帮助提高 RSA 性能：到目前为止，广告客户的点击份额增加了 9.4%，点击份额增加了 8.3%。启用自动生成的资产时，点击率增加了 %。1

您可以选择不需要的自动生成的标题和描述。

利用您现有的 RSA 资产创建多媒体广告 (MMA)，简化您的广告创建体验。

多媒体广告通过展示您的品牌和产品的大型视觉图像吸引消费者的注意力，从而提高知名度、兴趣和销量。为此，请选择一个自适应搜索广告，点击“编辑”，然后选择“将自适应搜索广告复制到多媒体广告”。

将图像、徽标和公司名称添加到新的多媒体广告中。

3. Logo extensions现已可用

好消息：广告商现在可以使用我们最新版本的徽标扩展将其徽标添加到广告中！这些附加信息可以与其他广告附加信息一起显示，并且可以在帐户、广告系列或广告组级别关联。

参阅新的帮助页面徽标扩展：向您的广告添加徽标 (microsoft.com)在新窗口中打开

4. 新客户数据平台集成 - Hightouch 和 Zapier

现在可以使用 Hightouch 和 Zapier 的新客户数据平台集成来设置和管理 Microsoft Advertising 上的客户匹配。

有关客户匹配的详细信息，请访问客户匹配：使用您自己的数据查找客户 (microsoft.com)在新窗口中打开

2023-08更新：

1. 微软广告效果最大化广告现已进入公开测试版

你听说过吗？我们现在推出了效果最大化广告系列 (PMAX)，这是一种新的广告系列类型，可通过自动化完善您的创意和定位组合。目前，该功能已针对全球所有广告商进行公开测试。

2. UET Web Insights 推出计划变更

将于 2023 年 7 月 25 日星期二开始向所有广告商开放。根据试点情况，此功能将在后续自动启用。

3. 有兴趣启用 UET Insights

通过导航到工具 > UET标签来访问 UET 仪表板，然后选择左侧导航栏上的相关标签。在标题摘要栏中，将鼠标悬停在“UET Insights”设置上，然后选择铅笔图标。在弹出窗口中（参见图片），打开 UET Insights Enabled 并选择“保存”。如果您想为多个标签启用 UET Insights，请访问“所有标签”页面，选择相关标签，然后选择“启用 Insights”。

从 Microsoft Advertising 平台轻松启用和禁用 UET Insights。

4. 智能广告系列帐户的免代码转化

全球范围内所有 Microsoft Advertising Smart Campaigns 都可以选择设置无代码转化目标

要启用此功能，请选中目标设置工作流程中的**启用 Microsoft Clarity框。**除了转化跟踪之外，您还可以启用免费的Microsoft Clarity Insights。有了这个，您将能够更好地了解点击后用户行为和登陆页面上的参与度，例如会话记录、即时热图和强大的分析仪表板。

5. Microsoft Audience Ads 将在另外 58 个市场/地区推出

我们不断开拓更多的 Microsoft Advertising 市场，并且我们很高兴与您分享更多 Microsoft Audience Ads 的市场（从 8 月中旬开始）：

**美洲：**安圭拉、安提瓜和巴布达、巴巴多斯、伯利兹、百慕大、英属维尔京群岛、格陵兰、格林纳达、瓜德罗普岛、牙买加、圣基茨和尼维斯、圣卢西亚、圣皮埃尔和密克隆群岛、圣文森特和格林纳丁斯、苏里南、土耳其和凯科斯群岛、美属维尔京群岛。

**亚太地区：**美属萨摩亚、圣诞岛、科科斯（基林）群岛、库克群岛、澳门、马绍尔群岛、瑙鲁、纽埃、诺福克岛、北马里亚纳群岛、帕劳、皮特凯恩群岛、萨摩亚、托克劳、汤加、图瓦卢、瓦努阿图、瓦利斯群岛和富图纳群岛。

**非洲：**贝宁、博茨瓦纳、布基纳法索、喀麦隆、中非共和国、科摩罗、科特迪瓦、吉布提、厄立特里亚、法罗群岛、加蓬、直布罗陀、几内亚比绍、马约特岛、尼日尔、刚果共和国、卢旺达、塞内加尔、塞拉利昂利昂、索马里、突尼斯、乌干达、赞比亚。
